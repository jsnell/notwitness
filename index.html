About

Some experimental puzzles in the style of (but not associated with) The Witness. This builds on the concepts and symbols from that game, so there is little point in trying these puzzles if you have not played it. Code on GitHub.

Controls

Arrow keys / WASD to steer. Hold down shift to increase speed. Space to reset puzzle. Page up / page down to move between puzzles.
